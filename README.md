# HH Parser вакансий на Python

Веб-приложение для сбора Python-вакансий с hh.ru, сохранения в SQLite и просмотра в удобном интерфейсе.

## Что умеет

- Парсит вакансии по запросам:
  - `Python разработчик`
  - `Python программист`
  - `Python developer`
  - `Data Engineer Python`
- Сохраняет данные в SQLite (`vacancies.db`).
- Показывает вакансии с фильтрами:
  - по роли,
  - по требуемому опыту,
  - по формату работы,
  - текстовый поиск.
- Показывает зарплату, если она есть в выдаче hh.ru.
- Ведет счетчик **новых вакансий за текущий запуск обновления**.
- Поддерживает **избранное**:
  - кнопка `В избранное` у вакансии,
  - хранение флага в БД,
  - отдельная страница `/favorites`.
- Есть страница **аналитики** `/analytics`:
  - метрики,
  - графики по ролям/опыту/формату,
  - топ компаний,
  - динамика добавления по дням.

## Технологии

- Python 3
- Flask
- SQLite
- Requests + BeautifulSoup4
- Bootstrap + Chart.js

## Быстрый старт

1. Создайте и активируйте виртуальное окружение:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Запустите приложение:

```bash
python3 app.py
```

4. Откройте в браузере:

`http://127.0.0.1:5000`

5. Нажмите кнопку **"Обновить вакансии из hh.ru"**.

## Основные страницы

- `/` — список вакансий и фильтры
- `/favorites` — избранные вакансии
- `/analytics` — аналитика по базе

## Структура проекта

- `parser_hh.py` — парсинг вакансий с hh.ru (HTML-выдача)
- `database.py` — схема БД и SQL-операции
- `app.py` — Flask-роуты и серверная логика
- `templates/index.html` — главная страница
- `templates/favorites.html` — избранные вакансии
- `templates/analytics.html` — страница аналитики
